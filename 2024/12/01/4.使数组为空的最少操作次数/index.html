4.使数组为空的最少操作次数

题目

给你一个下标从 0 开始的正整数数组 nums 。 你可以对数组执行以下两种操作任意次 ：

  • 从数组中选择 两个 值 相等 的元素，并将它们从数组中 删除 。
  • 从数组中选择 三个 值 相等 的元素，并将它们从数组中 删除 。
  • 请你返回使数组为空的 最少 操作次数，如果无法达成，请返回 -1 。

示例 1：

输入：nums = [2,3,3,2,2,4,2,3,4]
输出：4
解释：我们可以执行以下操作使数组为空：

  • 对下标为 0 和 3 的元素执行第一种操作，得到 nums = [3,3,2,4,2,3,4] 。
  • 对下标为 2 和 4 的元素执行第一种操作，得到 nums = [3,3,4,3,4] 。
  • 对下标为 0 ，1 和 3 的元素执行第二种操作，得到 nums = [4,4] 。
  • 对下标为 0 和 1 的元素执行第一种操作，得到 nums = [] 。
    至少需要 4 步操作使数组为空。

示例 2：

输入：nums = [2,1,2,2,3,3]
输出：-1
解释：无法使数组为空。

提示：
2 <= nums.length <= 105
1 <= nums[i] <= 106

关于unordered_map

这段代码使用了 unordered_map 来统计数组 nums 中每个元素出现的次数。具体解释如下：

  • unordered_map<int, int> cnt;：定义了一个无序映射 cnt，其中键（key）是 int 类型，表示数组中的元素；值（value）也是 int 类型，表示该元素出现的次数。
  • for (int x : nums)：这是一个范围基于的 for 循环，遍历 nums 数组中的每一个元素，并将当前元素的值赋给变量 x。
  • cnt[x]++;：对于每个元素 x，在 cnt 映射中对应的值加 1。如果 x 还没有出现在 cnt 中，则 cnt[x] 默认初始化为 0，然后加 1。

这样，循环结束后，cnt 映射中就存储了 nums 数组中每个元素出现的次数。

例如：
- 如果 nums = {1, 2, 3, 2, 1}，那么执行这段代码后，cnt 映射将会是 { {1, 2}, {2, 2}, {3, 1} }，表示数字 1 出现了 2 次，数字 2 出现了 2 次，数字 3 出现了 1 次。
