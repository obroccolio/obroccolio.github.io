n 皇后问题

回溯算法

按照国际象棋的规则，皇后可以攻击与之在同一行、同一列或同一斜线上的棋子。n 皇后问题研究的是如何将 n 个皇后放置在一个 n×n 的棋盘上，并且使皇后彼此之间不能相互攻击。

要求

给定一个整数 n，返回所有不同的 n 皇后问题的解决方案。每一种解法包含一个不同的 n 皇后问题的棋子放置方案，其中 ‘Q’ 和 ‘.’ 分别代表了皇后和空位。
